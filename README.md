# i-game-resource

## 简介

`i-game-resource` 是一个用于归档和发布独立 HTML 页面的仓库。这里存放的页面多为静态资源，不绑定特定域名或网站，适用于浏览、参考或直接使用。

本仓库旨在提供清晰、整洁的 HTML 页面存档，方便开发者或爱好者查阅与获取。

## 目录说明

```
i-game-resource/
├── pages/          # 存放独立 HTML 页面文件
├── assets/         # 页面引用的资源文件（如图片、样式、脚本）
└── README.md       # 本说明文件
```

- `pages/`：每个 HTML 文件代表一个独立的页面，命名简洁，便于识别。
- `assets/`：与页面配套的资源文件，按需组织，不强制统一格式。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面内容可能包含简单的交互功能，但无后端依赖。
- 归档时尽量保持原始结构与样式，不进行额外修改。
- 若页面引用外部资源（如字体库、CDN 库），会在页面头部注明来源。

## 维护说明

- 本仓库由维护者不定期更新，新增或更新页面会通过提交记录体现。
- 欢迎提交 Issue 或 Pull Request 建议新增归档页面，但需确保内容符合通用性及合规性。
- 仓库不提供在线托管服务，所有页面需自行部署或本地使用。
- 维护者保留对仓库内容的整理与清理权利。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 进入 `pages/` 目录，选择需要的 HTML 文件。
3. 在浏览器中打开该文件即可查看。
